# Li-Po Charger

小容量リポバッテリー充電器

## 仕様

- 電源入力 (5V)
    - USB Type-C
    - 2.54mm ピンヘッダ
- 充電電圧・電圧 (1個あたり)
    - 4.2V / 50mA

USB Type-Cとピンヘッダから同時に入力された場合、ピンヘッダからの入力を優先する

充電中赤色LED点灯、充電完了時に消灯
